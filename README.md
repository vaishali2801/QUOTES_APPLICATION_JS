ðŸ”§ Core Functionality

The quotes slider works on the following logic:

Quotes, images, and authors are stored as objects inside an array

An index variable tracks the currently active quote

Clicking Next increases the index

Clicking Previous decreases the index

When the last quote is reached, the slider loops back to the first

When navigating before the first quote, it jumps to the last

Auto-play changes quotes at a fixed time interval

Manual navigation resets the auto-play timer

This ensures smooth, infinite, and user-friendly quote navigation.
